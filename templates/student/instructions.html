Computer Based Test (CBT)
Candidate Guidelines
  1. The examination will be conducted in Computer Based Test (CBT) mode.
  2. The countdown timer displayed on the top-right corner shows the remaining time. When the timer reaches zero, the exam will be submitted automatically.
  3. Do not refresh the page or use browser back/forward buttons during the examination.
  4. Each question has four options (A, B, C, D). Only one option is correct.
  5. Use Save & Next to save your response and move to the next question.
  6. Use Mark for Review if you want to revisit the question later.
  7. Use Clear Response to remove the selected answer.
  8. You may navigate between questions using the Question Palette.
  9. Answers marked for review will be considered for evaluation only if they are answered.
  10. Any unfair means may lead to disqualification.
